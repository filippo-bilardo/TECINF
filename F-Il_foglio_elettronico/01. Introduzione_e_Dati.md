### Componenti Principali

#### 1. **Cella**
- UnitÃ  base del foglio di calcolo
- Identificata da **indirizzo**: lettera colonna + numero riga
- Esempio: `A1`, `B5`, `Z100`
- PuÃ² contenere: testo, numeri, formule, date

#### 2. **Riga**
- Sequenza orizzontale di celle
- Identificata da un **numero** (1, 2, 3, ...)
- Limite: ~1.000.000 di righe (varia per software)

#### 3. **Colonna**
- Sequenza verticale di celle
- Identificata da una **lettera** (A, B, C, ..., Z, AA, AB, ...)
- Limite: 
  - Excel: 16.384 colonne (A-XFD)
  - Google Sheets: 18.278 colonne

#### 4. **Intervallo (Range)**
- Gruppo di celle contigue
- Formato: `CellaPrimaAngolo:CellaOpposta`
- Esempi:
  - `A1:A10` â†’ colonna A, righe 1-10
  - `A1:C1` â†’ riga 1, colonne A-C
  - `A1:C10` â†’ rettangolo 3Ã—10 celle

#### 5. **Foglio (Sheet)**
- Pagina singola del documento
- Contiene griglia di celle
- Un file puÃ² avere piÃ¹ fogli (workbook)

#### 6. **Cartella di Lavoro (Workbook)**
- File completo
- Contiene uno o piÃ¹ fogli
- Estensioni: `.xlsx` (Excel), `.ods` (LibreOffice), `.gsheet` (Google)
