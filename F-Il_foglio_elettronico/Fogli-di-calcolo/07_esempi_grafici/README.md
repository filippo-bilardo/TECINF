# Esempi Pratici - Grafici e Visualizzazioni

Questa cartella contiene esempi pratici da utilizzare in Google Fogli per esercitarsi con i diversi tipi di grafici.

## Indice Esempi

1. **01_vendite_mensili.md** - Grafico a colonne e linee per trend vendite
2. **02_confronto_anni.md** - Grafico a colonne raggruppate per confronto multi-anno
3. **03_composizione_ricavi.md** - Grafico a torta per analisi composizione
4. **04_dashboard_sparkline.md** - Dashboard con mini-grafici sparkline
5. **05_grafico_combinato.md** - Grafico combinato con doppio asse
6. **06_analisi_correlazione.md** - Grafico a dispersione per correlazioni
7. **07_performance_kpi.md** - Dashboard KPI con gauge e indicatori
8. **08_analisi_temporale.md** - Grafico ad area per evoluzione temporale
9. **09_ranking_prodotti.md** - Grafico a barre per classifiche
10. **10_waterfall_budget.md** - Analisi waterfall per variazioni budget

## Come Usare gli Esempi

1. Apri un nuovo Google Foglio
2. Copia i dati dall'esempio scelto
3. Incolla nel foglio Google
4. Segui le istruzioni passo-passo per creare il grafico
5. Sperimenta con le personalizzazioni suggerite

## Suggerimenti

- Ogni esempio include i dati già pronti
- Le istruzioni sono dettagliate e guidate
- Prova a modificare i dati per vedere come cambia il grafico
- Combina più tecniche per creare dashboard complete
